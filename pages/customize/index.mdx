# Repository customizations

## Simple customizations

The following parts of the repository can be customized without the need to modify the code:

| Category | Customization | Documentation |
|:---|:---|:---|
| **Branding** | change name, logo, and color palette. | [Simple branding](/customize/branding/simple) |
| **Record metadata** | add new fields to the model | [Custom fields](/customize/record/custom_fields) |
| **Record metadata** | change the order of the fields on the detail page | [Detail page template](/customize/record/detail) |
| **Record metadata** | change the order of the fields on the deposit page. | [Deposit page template](/customize/record/deposit) |
| **Vocabularies** | initial values of vocabularies | [Fixtures](/customize/vocabularies) |
| **Translations** | change translations | [Translations](/customize/translations) |


## Intermediary customization

The following parts of the repository can be customized by modifying the code:

| Category | Customization | Documentation |
|:---|:---|:---|
| **Branding** | change the layout of pages | [Page templates](/customize/branding/layout) |
| **Record metadata** | change the display of records on the list page | [Record List Item](/customize/record/list) |
| **Record metadata** | add a whole new model | [Adding new model](/customize/record/new_model) |
| **Record metadata** | add complex metadata fields | [Complex metadata fields](/customize/record/complex_fields) |
| **Record metadata** | add complex validation of the fields | [Complex validation](/customize/record/validation) |
