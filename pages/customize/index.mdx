# Repository customizations

## Simple customizations

The following parts of the repository can be customized without the need to modify the code and without the need to deep dive into some advanced Invenio concepts.

<Cards>
<Card title='Branding' arrow href="/customize/branding/theme">Change site name, brand logo, color palette, fonts and styling.</Card>
<Card title='Custom fields' arrow href="/customize/record/custom_fields">Add custom metadata fields to your records.</Card>
<Card title='Vocabularies' arrow href="/customize/vocabularies/initial_data">Provide custom initial data for controlled vocabularies.</Card>
<Card title='Translations' arrow href="/customize/translations">Translate into your language, or customize existing translations.</Card>
<Card title='Record landing page' arrow href="/customize/record/detail">Change the order of metadata fields displayed on the record detail page.</Card>
<Card title='Record deposition' arrow href="/customize/record/deposit">Change the order of metadata fields displayed in the record deposition form.</Card>
<Card title='Record workflows' arrow href="/customize/workflows">Customize record workflows and approval processes.</Card>
</Cards>
