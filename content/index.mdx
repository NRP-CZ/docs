# Czech National Repository Platform

<Callout type="warning" emoji="⚠️">
  This documentation is a work in progress and may contain inaccuracies or omissions.
  If you have any questions, please contact us at nrp-support@cesnet.cz.
</Callout>

Czech National Repository Platform is a set of infrastructure services
intended to support the operation of digital repositories in the Czech Republic.

The platform hosts repositories built using several frameworks, including:

* Invenio Platform - a digital repository platform based on Invenio RDM, provided by CESNET
* Clarin DSpace - a digital repository platform based on DSpace, provided by LINDAT/CLARIN
* ARL - a digital repository platform used on the Czech Academy of Sciences

This document focuses mostly on the Invenio Platform. For information about the other platforms,
please refer to their respective documentation pages.

## Getting help

If you need assistance, please reach out to our support team at cesnet-invenio@eosc.cz.
