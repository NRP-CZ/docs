# Fonts <Difficulty level="easy" />

This section explains how to customize typography in an NRP Invenio repository, following the InvenioRDM theming guidelines described in the [InvenioRDM Fonts Documentation](https://inveniordm.docs.cern.ch/operate/customize/look-and-feel/font/). You can use any custom font as long as you provide the files and declare them in your LESS theme.

<Steps>

### Provide font assets

Place your font files inside your repository-level assets folder.  
In NRP Invenio, this follows the InvenioRDM structure:

<FileTree>
  <FileTree.Folder name="myrepo" defaultOpen>
    <FileTree.Folder name="assets" defaultOpen>
      <FileTree.Folder name="semantic-ui" defaultOpen>
        <FileTree.Folder name="less" defaultOpen>
          <FileTree.Folder name="fonts" defaultOpen>
            <FileTree.File name="my-font.ttf" />
            <FileTree.File name="my-heading-font.ttf" />
          </FileTree.Folder>
        </FileTree.Folder>
      </FileTree.Folder>
    </FileTree.Folder>
  </FileTree.Folder>
</FileTree>

```shell copy
cp {my-font.ttf,my-heading-font.ttf} myrepo/assets/semantic-ui/less/fonts/
```

### Configure [`@font-face`](https://developer.mozilla.org/en-US/docs/Web/CSS/@font-face) rules
Define the fonts in your global stylesheet overrides, typically in your [Global stylesheet](/customize/repository_ui/branding/styling#global-stylesheet) file:

```less filename="/assets/semantic-ui/less/site/globals/site.overrides" copy
@font-face {
  font-family: 'MyFont';
  src: url("~@less/fonts/my-font.ttf");
}

@font-face {
  font-family: 'MyHeading';
  src: url("~@less/fonts/my-heading-font.ttf");
}
```

### Apply fonts in your theme
You can now apply your custom font to any global or component-level styles.
Global typography override (for the entire site):
```less
html {
  font-family: "MyFont";
}
```

Component-specific typography (e.g., Semantic UI Headers):
```less
.ui.header {
  font-family: "MyHeading";
}
```
</Steps>
