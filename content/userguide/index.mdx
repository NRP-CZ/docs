## Command line interface

For advanced users, the command line interface can be used to interact with the repository. User can:

* search records in the repository
* download records and their datasets
* upload new records and datasets or update existing ones
* publish records and datasets
* approve publishing requests
* assign DOIs to records and datasets

See the [CLI documentation](userguide/commandline) for more details.

## Python client library

The NRP Python client library allows programmatic interaction with Invenio NRP repositories. It provides both synchronous and asynchronous APIs for managing records, files, and requests.

See the [Python client library documentation](userguide/python_library/) for more details.
